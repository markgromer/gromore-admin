Set Up Your Google Business Profile Connection

To see your profile details, you need a Google Place ID and a Google Maps API key configured in your business settings.

Quick setup:
  1. Find your Place ID at Google's Place ID Finder
  2. Your Maps API key should already be set up. If not, create one in Google Cloud Console
  3. Make sure the Places API (New) is enabled in your Google Cloud project
  4. Enter both values in your business settings

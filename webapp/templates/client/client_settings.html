How to get an Application Password
1. Log into your WordPress admin (yourdomain.com/wp-admin)
2. Go to Users > Profile
3. Scroll down to Application Passwords
4. Enter a name like "GroMore" and click Add New Application Password
5. Copy the generated password and paste it above
Note: Your site must use HTTPS and have the REST API enabled (most sites do by default).

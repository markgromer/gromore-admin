Template Guide

Templates are reusable HTML/CSS sections that the site builder can inject into generated pages.

Categories:
  • Hero - Top-of-page hero sections
  • Section - Content sections (trust strips, CTA bands)
  • Widget - Standalone elements (pricing cards, testimonial walls)
  • Footer - Footer variations
  • Navigation - Header/nav patterns
  • Page Shell - A full page scaffold with a {% raw %}{{page_content}}{% endraw %} slot Warren fills for that page type
Page Types:

Comma-separated list of page types this template can be used on. Leave blank for all pages.

Site Template Guide

A full site template is a kit that combines one theme with a curated set of builder templates. Brands choose one kit during intake, and Warren uses that kit instead of the global defaults.

  • Attach one theme preset for colors, fonts, and overall styling.
  • Attach only the templates that belong in this site kit.
  • Use page shell templates when you want Warren to fill a deterministic layout instead of inventing a fresh structure.
  • Keep prompt notes short and specific so they meaningfully steer generation.

Theme Guide

Themes define the visual identity passed to the site builder. Colors, fonts, button styles, and layout preferences are included in the generation prompt so AI-generated pages match the brand's look.

Button Styles:
  • rounded - Soft rounded corners
  • pill - Full pill shape
  • square - Sharp corners
Layout Styles:
  • modern - Clean, whitespace-heavy
  • classic - Traditional, structured
  • bold - Large type, high contrast
  • minimal - Ultra-clean, simple

Prompt Override Guide

Overrides let you customize or extend the AI prompts without changing code. They layer on top of the built-in prompts.

Sections:
  • User Prompt - Appended to the page-specific prompt. Use for extra instructions, industry-specific rules, or brand voice guidelines.
  • System Message Override - Replaces the default system message entirely. Use with caution.
  • Extra Rules - Appended to GLOBAL_RULES for this page type specifically.
Special Page Types:
  • global_rules - Rules that apply to ALL page types
  • system_message - Override the system persona for all pages
